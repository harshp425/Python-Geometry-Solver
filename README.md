# Python Geomtry Solver

This is an interactive python program that allows users to solve select geometry problems in seconds. Thr program is written completely in python and leverages the Math library, classes, and control structures such as while loops and conditional statements.

The program offers solutions for a variety of geometry problems detailed in the image below...

## How to Run
This geometry solver runs completely on the terminal and so the code in the Main.py file can be placed into any IDE and run either in the machine's own terminal or in an integrated terminal. 

Once initialized, the user must type in the number corresponding to the geometry problem they wish the program to solve. 

<img width="754" alt="Screenshot 2024-01-07 at 5 22 19 PM" src="https://github.com/harshp425/Python-Geomtry-Solver/assets/126726290/48aa55ad-7606-4283-94da-03bb55b234df">



They will then be prompted to enter specific information based on the type of problem chosen.

<img width="757" alt="Screenshot 2024-01-07 at 5 24 32 PM" src="https://github.com/harshp425/Python-Geomtry-Solver/assets/126726290/7ec707dc-7e42-4250-a664-d95c75f96413">


After the problem specific information has been filled, upon pressing  `enter`, the program will output the answer before automatically returning to the menu.
